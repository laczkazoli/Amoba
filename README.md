# Amoba game

This is a two player Amoba game. You need to have five pair next to each other

## Running

```sh
mvn package
java -jar target/Amoba-0.1-jar-with-dependencies.jar
```
or
```sh
mvn clean
mvn compile
mvn install
java -jar target/Amoba-0.1-jar-with-dependencies.jar
```
## Site

```sh
mvn clean
mvn compile
mvn install
mvn site
```
